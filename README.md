# Tensorflow Smile Detection

Detecting smile using Lenet.

## Setup
Generate Virtual environment
```bash
python3 -m venv ./venv
```
Enter environment
```bash
source venv/bin/activate
```
Install required libraries
```bash
pip install -r requirements.txt
```
Train model
```bash
python3 train.py -d dataset
```
Test model
```
python3 test.py 
```
Smile!
